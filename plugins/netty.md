# 什么是Netty
Netty 是一款基于异步的事件驱动的、快速开发高性能的网络应用程序Java框架，支持快速地开发可维护的高性能的面向协议的服务器和客户端。它封装了网络编程的复杂性，使网络编程和 Web 技术的最新进展能够被比以往更广泛的开发人员接触到。
关键字：高性能事件驱动、异步非阻塞、基于NIO的客户端、服务器端、稳定性和可伸缩性

# Netty使用场景
* 高性能通信领域（游戏、分布式）
* 多线程并发领域
* 异步通信领域
